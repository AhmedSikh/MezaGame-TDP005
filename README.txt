In order to run the game:
============================================

Make sure you have the following installed before compiling.
============================================
SFML
Cmake

============================================
Installation Instructions
============================================

-Navigate to source code.
-Command: 'cmake .'(don't forgett the dot).
-Command: 'make'

============================================
Run
============================================

-Command: './sfml'

============================================

Controllers
============================================

Menu:
============================================
Up: w
Down: s
Chose: Enter

============================================

Game:
=============================================
Up: w
Down: s
Left: a
Right: d
Exit: q or Escape
